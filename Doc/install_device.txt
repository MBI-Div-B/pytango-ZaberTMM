Installieren des Devices mittels "Jive"
#######################################

1. Installieren
---------------

Anlegen eines neuen Devices:

Server (Servername/Instance) : ZaberTMM/hhg
Class: ZaberTMM
Device: hhg/ZaberTMM/1


1.2. Konfiguration
-------------------

Die Properties:
Name: Port		Value: com4 		# serieller Port (com1, com2 .., /dev/tty, /dev/ttyUSB0 ..  )
	
1.3. Starten
---------------

# python ZaberTMM.py hhg
